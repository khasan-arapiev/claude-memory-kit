# Website / Marketing Project Type

## Extra folder

Adds `copy/` folder for voice, messaging, landing page copy.

```
project/
└── copy/
    ├── voice-and-tone.md
    ├── headlines.md
    └── ctas.md
```

## CLAUDE.md additions

Add this section to the project's CLAUDE.md after Task Routing:

```markdown
### Working on copy or messaging
1. Read `docs/copy/` files (or `project/copy/` for production copy)
2. Always apply parent writing rules
```

## Initial doc stubs

When scaffolding a website project, create these starter docs in `docs/context/`:
- `BRAND-IDENTITY.md` (placeholder)
- `TARGET-AUDIENCE.md` (placeholder)
- `VALUE-PROPOSITION.md` (placeholder)
