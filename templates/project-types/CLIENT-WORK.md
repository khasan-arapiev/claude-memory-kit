# Client Work Project Type

## Extra folder

Adds `deliverables/` folder for client-facing outputs and reports.

```
project/
└── deliverables/
    ├── reports/
    ├── presentations/
    └── exports/
```

## CLAUDE.md additions

Add this section to the project's CLAUDE.md after Task Routing:

```markdown
### Working on client deliverables
1. Read `docs/context/CLIENT-BRIEF.md` for scope and goals
2. Read `docs/workflows/DELIVERY-PROCESS.md` for handoff steps
```

## Initial doc stubs

When scaffolding a client work project, create these starter docs:
- `docs/context/CLIENT-BRIEF.md` (placeholder)
- `docs/context/CLIENT-CONTACTS.md` (placeholder)
- `docs/workflows/DELIVERY-PROCESS.md` (placeholder)
